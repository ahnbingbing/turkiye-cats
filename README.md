# 고양이 공화국, 튀르키예 🐈

튀르키예 여행에서 만난 길고양이 이야기 — 인터랙티브 웹 발표 자료 (by bb.tan).

## 보기
- 브라우저에서 **`index.html`** 을 열면 바로 실행돼요. (로컬에서 더블클릭해도 동작)
- 온라인 공유: 이 폴더를 GitHub repo에 올리고 **Pages** 를 켜면 링크로 공유돼요.

## GitHub Pages 배포
1. 이 폴더(`index.html` + `assets/` + `.nojekyll`)를 repo에 push
2. **Settings → Pages → Source: Deploy from a branch → `main` / `(root)`**
3. `https://<아이디>.github.io/<repo>/` 로 접속

## 구성
- `index.html` — 발표 화면 전체 (HTML/CSS/JS 단일 파일)
- `assets/img/` — 사진, `assets/img/fish/` — 부둣가 컷, `assets/vid/` — 영상 클립
- 외부 리소스: Google Fonts, 위키미디어(글리·톰빌리), YouTube(케디 예고편) — 인터넷 연결 필요

## 조작
- 좌우 화살표 / 클릭(화면 좌·우) / 방향키 — 슬라이드 이동
- 우상단 **슬라이드·목차·갤러리** 탭, `?` 호버 설명, ‘자세히/더 보기’ 패널
